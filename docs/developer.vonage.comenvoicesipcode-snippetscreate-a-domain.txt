Create a Domain

In the below code snippet, you see how to create a domain for the Programmable SIP API.

Example

Replace the following variables in the example code:

Key	Description
SIP_DOMAIN	

The name of your SIP Domain.


VONAGE_APPLICATION_ID	

The Vonage Application ID.


VONAGE_API_KEY	

Your Vonage API key (see it on your dashboard).


VONAGE_API_SECRET	

Your Vonage API secret (also available on your dashboard).

Prerequisites
Write the code

Add the following to create-domain.sh:

MY_IP_ADDRESS=$(curl ifconfig.co)

curl -X POST -H "Content-Type: application/json" \
    -d '{"name": "'$SIP_DOMAIN'", "application_id": "'$VONAGE_APPLICATION_ID'", "acl": ["'$MY_IP_ADDRESS'"]}' \
    https://api.nexmo.com/v1/psip -u $VONAGE_API_KEY:$VONAGE_API_SECRET
Copy

View full source

Run your code

Save this file to your machine and run it:

Copy
bash create-domain.sh
