Delete a Domain

In the below code snippet, you see how to delete a domain for the Programmable SIP API.

Example

Replace the following variables in the example code:

Key	Description
SIP_DOMAIN	

The name of your SIP Domain.


VONAGE_API_KEY	

Your Vonage API key (see it on your dashboard).


VONAGE_API_SECRET	

Your Vonage API secret (also available on your dashboard).

Prerequisites
Write the code

Add the following to delete-domain.sh:

curl -X DELETE https://api.nexmo.com/v1/psip/$SIP_DOMAIN -u $VONAGE_API_KEY:$VONAGE_API_SECRET
Copy

View full source

Run your code

Save this file to your machine and run it:

Copy
bash delete-domain.sh
